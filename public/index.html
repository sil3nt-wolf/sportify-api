Usage Documentation


  
  

    

      GET
      /api/token
      Get a live Spotify access token
      ▶
    

    

      
Returns the current anonymous Spotify web-player access token. Use this token in any direct Spotify API call.

      
Request

      
GET /api/token

      
Response

      
{
  "provider": "CASPER TECH",
  "creator": "TRABY CASPER",
  "success": true,
  "note": "Anonymous Spotify web-player token. Valid for ~1 hour.",
  "access_token": "BQDxxxxxx...",
  "token_type": "Bearer",
  "usage": {
    "search": "GET https://api.spotify.com/v1/search?q=Faded&type=track&limit=10",
    "track": "GET https://api.spotify.com/v1/tracks/{id}",
    "album": "GET https://api.spotify.com/v1/albums/{id}",
    "playlist": "GET https://api.spotify.com/v1/playlists/{id}",
    "artist": "GET https://api.spotify.com/v1/artists/{id}"
  }
}

      
Using the token with Spotify directly

      
curl -H "Authorization: Bearer BQDxxxxxx..." \
  "https://api.spotify.com/v1/search?q=Faded&type=track&limit=5"

    

  


  
  

    

      GET
      /api/search
      Search Spotify catalogue
      ▶
    

    

      
Search across Spotify's full catalogue. Returns matching results for the given type.

      

        
ParameterTypeDescription

        
qstring · requiredSearch query e.g. Faded, Drake, Chill Vibes

        
typestring · requiredComma-separated: track, album, artist, playlist, episode, show

        
limitnumber · optionalResults to return. Default: 10. Max: 50

        
offsetnumber · optionalPagination offset. Default: 0

      

      
Example — search for a track

      
GET /api/search?q=Faded&type=track&limit=5

      
Example — search for an artist

      
GET /api/search?q=Drake&type=artist&limit=3

      
Example — multiple types

      
GET /api/search?q=Chill&type=track,album,playlist&limit=5

    

  


  
  

    

      GET
      /api/track/:id
      Get track details by Spotify ID
      ▶
    

    

      
Returns full track metadata including name, artists, album, duration, and audio preview URL.

      

        
ParameterTypeDescription

        
idstring · requiredSpotify track ID (22-character string from the Spotify URL)

      

      
Example

      
GET /api/track/3n3Ppam7vgaVa1iaRUIOKE

      
The track ID is the last part of a Spotify URL: open.spotify.com/track/3n3Ppam7vgaVa1iaRUIOKE

    

  


  
  

    

      GET
      /api/album/:id
      Get album details and track listing
      ▶
    

    

      
Returns album metadata including name, artists, release date, total tracks, and the full track list.

      

        
ParameterTypeDescription

        
idstring · requiredSpotify album ID from the album URL

      

      
Example

      
GET /api/album/1weenld61qoidwYuZ1GESA

    

  


  
  

    

      GET
      /api/playlist/:id
      Get playlist info and all tracks
      ▶
    

    

      
Returns playlist metadata, owner, description, and the full list of tracks.

      

        
ParameterTypeDescription

        
idstring · requiredSpotify playlist ID from the playlist URL

      

      
Example — Spotify Global Top 50

      
GET /api/playlist/37i9dQZEVXbMDoHDwVN2tF

    

  


  
  

    

      GET
      /api/artist/:id
      Get artist profile and metadata
      ▶
    

    

      
Returns artist name, genres, follower count, popularity score, and profile images.

      

        
ParameterTypeDescription

        
idstring · requiredSpotify artist ID from the artist URL

      

      
Example — Drake

      
GET /api/artist/3TVXtAsR1Inumwj472S9r4

    

  


  
  

    

      GET
      /api/artist/:id/top-tracks
      Get artist top tracks by market
      ▶
    

    

      
Returns the artist's top 10 tracks in a given market, with track metadata and audio preview URLs.

      

        
ParameterTypeDescription

        
idstring · requiredSpotify artist ID

        
marketstring · optionalISO 3166-1 alpha-2 country code. Default: US

      

      
Example

      
GET /api/artist/3TVXtAsR1Inumwj472S9r4/top-tracks?market=US

      
Other markets

      
?market=GB   — United Kingdom
?market=KE   — Kenya
?market=NG   — Nigeria
?market=ZA   — South Africa
